# Pistanero Store

> E-commerce storefront built for a client — live at [pistanero.store](https://www.pistanero.store)

---

## Overview

**Pistanero** is a fully functional online store built and deployed for a client. The project covers the complete storefront experience — from product browsing and cart management to checkout.

## Features

- Product listings with search and filtering
- Shopping cart and checkout flow
- Responsive design across mobile and desktop
- Secure payment integration
- Order management

## Tech Stack


Frontend : React
Backend  : Supabase
Database : MySQL
Hosting : Namecheap


## Getting Started

```bash
# Clone the repo
git clone https://github.com/your-username/pistanero-store.git
cd pistanero-store

# Install dependencies
npm install

# Start the development server
npm run dev
```

## Environment Variables

Create a `.env` file in the root directory:

```env
# Example — update with your actual variables
DATABASE_URL=your_database_url
PAYMENT_SECRET_KEY=your_payment_key
```

## Deployment

The store is live at **[www.pistanero.store](https://www.pistanero.store)**.
